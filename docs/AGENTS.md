## Overview

ZENTR3 deploys **25 specialized AI agents**, each powered by **Google Gemini 2.5 Flash** with a dedicated system prompt. Together they form a complete startup-building swarm — from market research to smart contract deployment.

## Technical Implementation

All agents share the same infrastructure:

```typescript
// Agent system prompt lookup
const systemPrompt = AGENT_PROMPTS[body.agentRole] 
  || body.systemPrompt 
  || `You are the ${body.agentRole} agent.`;

// Gemini 2.5 Flash streaming
const stream = await ai.models.generateContentStream({
  model: "gemini-2.5-flash",
  contents: [...history, userMessage],
  config: {
    maxOutputTokens: 8192,
    systemInstruction: systemPrompt,
  },
});
```
